# Wordle - CS 2640 Final Project
## Group Members: Tyler Carrasco, Tiffany Dinh, Kendan Phan, Angela Salcido

## Purpose
This project is based on the popular online word guessing game, “Wordle,” using MIPS assembly. Its purpose is to stimulate the brain and allow users to engage in daily entertainment.

## How to Use
The program gives the user six attempts to correctly guess a randomly generated five-letter word. The user will input different letters and the program will output whether or not the letter is in the correct place, and if the letter is used in the word. When the six attempts are up, the user loses.
